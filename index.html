Real-world intelligent systems built from first principles — from sensor data to autonomous decisions.

Autonomous Vehicle Stack

A full autonomous driving pipeline built on Autoware Universe and ROS2. Implements localization, perception, prediction, planning, and control for real-world urban navigation scenarios.

LiDAR / Camera → Sensor Fusion → NDT Localization → Object Detection → Behavior Planning → MPC Controller
ROS2 Autoware C++ LiDAR NDT-SLAM

Depth Camera Pipeline

A real-time stereo and RGB-D depth estimation pipeline that generates dense 3D point clouds for robotic perception. Includes camera calibration, disparity estimation, and scene reconstruction modules.

RGB-D Camera → Calibration → Stereo Matching → Depth Map → PointCloud3D → Obstacle Map
OpenCV Python RealSense PCL ROS2

Exoplanet Prediction Platform

A machine learning platform for detecting exoplanet candidates from NASA Kepler telescope light curve data. Employs CNNs and transformer models to classify stellar brightness anomalies with 94% accuracy.

Kepler Data → Preprocessing → CNN Feature Extractor → Transformer Encoder → Classification → Dashboard
PyTorch Transformers NASA API Streamlit CUDA

Urban Encroachment Detection

A satellite & drone imagery analysis system that detects illegal urban encroachments on public land using temporal change detection and instance segmentation. Deployed for smart city governance initiatives.

Satellite / UAV → Geo-Registration → Change Detection → YOLO Segmentation → GIS Mapping → Alert System
YOLOv8 GDAL Satellite GIS FastAPI
